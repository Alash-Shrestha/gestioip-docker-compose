
#############################################
# Howto test the backup server installation #
#############################################


* Execute then commands from a terminal of your GestioIP server.

* Replace the 1.1.1.1 in the following commands with the IP address of
  your GestioIP server. Do not use localhost (127.0.0.1).

* Make sure that your firewall settings allow the requiered connections.

* Depending on your Linux distribution, command outputs may look slightly
  different.


Note for Fedora/Redhat/CentOS
If something is not working, disable SELinux for testing:
$ sudo setenforce 0


### TFTP server upload test ###

$ sudo touch /var/lib/tftpboot/test.conf
$ sudo chmod 777 /var/lib/tftpboot/test.conf 
$ echo hola > ./test.conf
$ tftp 1.1.1.1
tftp> put test.conf
tftp> quit
$ cat /var/lib/tftpboot/test.conf 
hola
$
